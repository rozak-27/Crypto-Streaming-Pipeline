Crypto Streaming Pipeline

A real-time crypto analytics pipeline built using modern data engineering tools.
This project streams live market data, processes it with Spark, stores it in PostgreSQL, and visualizes everything in Grafana dashboards.

What This Project Does
-Collects real-time crypto market data
-Streams data through Kafka
-Processes data using Spark Structured Streaming
-Stores processed data in PostgreSQL/TimescaleDB
-Visualizes live metrics and technical indicators in Grafana
-Runs scheduled data quality & transformation workflows using Airflow + dbt

Tech Stack
-Python
-Ap-ache Kafka
-Apache Spark
-PostgreSQL / TimescaleDB
-Grafana
-Prometheus
-Airflow
-dbt
-Docker Compose
